# Meridian University — Open Search

A design demo of an AI-powered site search experience for a university,
re-themed from the original healthcare version of
[open-search-v3-modal](https://github.com/uxpreview/open-search-v3-modal).

**Live preview:** https://uxpreview.github.io/open-search-university/

No build step — it's React + Babel Standalone loaded straight in the browser.
Open `index.html` (via any static server) and press <kbd>⌘K</kbd> / <kbd>Ctrl K</kbd>
or click **Search**.

## What's in the demo
- **University site shell** — a Meridian University home page plus a contextual
  Computer Science program page, each able to open the AI search modal.
- **Two chat-answer flows**
  - *Explore a major* — "Tell me about the Computer Science major" (overview,
    faculty, courses, outcomes, campus, related programs).
  - *Cost, aid & scholarships* — "How much does Meridian cost and what aid can I
    get?" (cost breakdown, sample aid package, how-aid-works steps,
    scholarships, financial-aid office, FAQ).
- **Compare two programs** — "Compare Computer Science and Data Science"
  (side-by-side table, overlap, how to choose).
- **Scope tabs** — pick a tab (Programs / Faculty / Campus / Cost & Aid /
  Admissions) to land an answer on a specific result type.
- **Degree Planner agent** — a 3-step guided wizard (interest → degree level →
  format) that matches you with degrees and hands off into an answer.
